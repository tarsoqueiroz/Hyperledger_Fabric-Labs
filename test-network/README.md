# test-network

## Pre-req

Installed products:

- [Virtualbox](https://www.virtualbox.org/manual/ch02.html)
- [Vagrant](https://www.vagrantup.com/docs/installation)

## Steps to run

- Clone the repo
- Go to test-network
- Building VM:

```shell
vagrant up
```

- Now, you need entering on VM and run the scripts:

```shell
vagrant ssh


```
